# Stonebridge Realty Group — Demo Website (Static)

This ZIP contains a complete, responsive static website for demo purposes.

## Brand system (included)
- **Logo:** `assets/img/logo.svg` (editable vector)
- **Icon/Favicon:** `assets/img/favicon.svg`
- **Font:** Google Fonts — Plus Jakarta Sans
- **Palette:** 
  - Ink #0B1F2A
  - Ocean #0F4C5C
  - Mint #2EC4B6
  - Sand #F4EFEA
  - Sun #FFB703

## Pages
- `index.html` (Home)
- `buy.html`
- `sell.html`
- `rent.html`
- `agents.html`
- `about.html`
- `contact.html` (mailto contact form)
- `disclosures.html` (wire fraud + fair housing)
- `privacy.html` (demo placeholder)

## Chatbot integration
A floating “Chat with Mia” button opens a modal placeholder. Replace the modal with your embedded widget script/iframe.

## How to run locally
Just open `index.html` in a browser, or serve the folder with any static server.

## Replace demo contact details
Edit the `brand` values directly in the HTML pages (search for the phone/email/address).

All names, addresses, and numbers are fictional.
