# PeakPoint Orthopedics & Sports Medicine — Demo Website (Fictional)

This is a static, demo-ready website for a fictional orthopedic & sports medicine clinic.

## Brand system
- **Primary font:** Inter (Google Fonts). Inter is open-source under SIL Open Font License.  
- **Color palette (CSS variables in `css/styles.css`):**
  - Ink / Navy: `#0B1F3A`
  - Deep Navy: `#07162B`
  - Teal: `#0EA5A4`
  - Teal Dark: `#0B7F7E`
  - Sky: `#EAF3FF`
  - Background: `#F6F8FB`

## Pages
- `index.html` — Home / conversion landing page
- `services.html` — Service overview
- `conditions.html` — Conditions treated (SEO-friendly)
- `patients.html` — New patient resources + urgent triage guidance
- `contact.html` — Contact details + message form (mailto)
- `privacy-policy.html` — Demo privacy policy

## How to run locally
Open `index.html` in a browser.

## How to deploy
Upload the folder contents to any static host (Netlify, Vercel static, Cloudflare Pages, S3, etc.).

## Notes
- The message form uses a `mailto:` fallback (no backend).
- All phone numbers, emails, addresses, and clinic details are fictional.
