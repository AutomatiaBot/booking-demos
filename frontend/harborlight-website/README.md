# Harborlight Counseling & Wellness (Fictional Demo Website)

This is a **static website** (HTML/CSS/JS) created for a demo therapy practice.

## How to run locally
- Open `index.html` directly in a browser, **or**
- Use a simple static server:
  - Python: `python -m http.server 8080`
  - Then visit: http://localhost:8080

## Notes
- The contact form is client-side only (no backend).
- All business details are fictional.
